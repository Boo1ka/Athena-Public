## 3. The "Vocabulary vs. Grammar" Heuristic

Use this mental model to separate "Information" from "Insight":

- **Vocabulary (Static)**: The "What". Definitions, syntax, tools. Scalable, low value. (e.g., "How to run a Facebook Ad")
- **Grammar (Dynamic)**: The "How". Timing, pattern recognition, judgment. Unscalable, high value. (e.g., "When to kill an ad set based on subtle cues")

**Rule**: We sell **Grammar**, not Vocabulary.

## 4. The "Insight Decay" Guard

> **Warning**: Today's "Unique Spin" is tomorrow's "Commodity Wisdom".

- **Review Cycle**: Every 6 months.
- **Action**: If a mechanism becomes mainstream, it is deprecated. We must invent the *next* frame.

## 5. Implementation in Content (The 3-Part Cluster)

1. **The Symptoms (Diagnosis)**: "You think you have [Common Problem], but you actually have [Root Cause]."
2. **The Mechanism (Theory)**: "Here is the technical/structural reason why the Common Solution fails."
3. **The Implementation (Proof)**: "Here is exactly how we solved it (N=1 Data)."
